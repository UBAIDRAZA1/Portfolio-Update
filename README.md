# Portfolio Setup Instructions

## Install Dependencies

Before running the application, you need to install the required dependencies:

1. Double-click on `install-dependencies.bat` to install resend package
2. Or run in command prompt:
   ```
   npm install
   ```

## Environment Variables

Make sure to set up your `.env.local` file with the following variables:

```
RESEND_API_KEY=your_actual_resend_api_key
PERSONAL_EMAIL=ubaidraza3657767@gmail.com
EMAIL_FROM=onboarding@resend.dev
```

## Running the Application

To run the application locally:
```
npm run dev
```

## Deploying to Vercel

When deploying to Vercel, make sure to add the same environment variables in your Vercel project settings.

## Troubleshooting

If you encounter any issues:
1. Make sure you have Node.js installed
2. Check that your API keys are correct
3. Verify that your email addresses are properly formatted"# Portfolio-Update"
